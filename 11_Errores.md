# ERRORES FRECUENTES:

| Señal                 | Qué significa       |
| --------------------- | ------------------- |
| `Running + 0/1 READY` | Readiness fallando  |
| `CrashLoopBackOff`    | App se cae          |
| `ImagePullBackOff`    | Imagen inválida     |
| `Unhealthy`           | Probe falló         |
| `Events`              | Fuente de la verdad |
