# Â¿Como entrar a un container?

**Primero, vemos los pods, luego entramos al container con el nombre del pod previamente obtenido de la lista**
```
kubectl get pods
kubectl exec -it us-service-example sh
ls -l
```
**Es como si entraramos a un sistema operativo, parecido a docker**

### Entraremos a un container y probaremos conexion, si esta llegando conexiona nuestra aplicacion, para esto ejecutamos los siguientes comandos
```
kubectl get pod
kubectl exec -it us-service-example sh
curl localhost
```
**Si esta todo bien, nos deberia dar alguna respuesta, de lo contrario, un error 400 o 500**
